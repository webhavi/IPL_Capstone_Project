# 🏏 IPL Data Analysis Project 📊

## 🚀 Project Overview

This project is an **Interactive IPL Data Analysis Dashboard & Exploratory Data Analysis (EDA)** built using Python and powerful data science libraries. The goal of this project is to uncover meaningful insights from IPL match data such as winning trends, toss impact, top performers, highest scores, best bowling figures, and team statistics. 📈🔥

Using **NumPy**, **Pandas**, **Matplotlib**, and **Seaborn**, I performed data cleaning, preprocessing, visualization, and statistical analysis to explore the dataset deeply.
